## 🏥 College Dispensary

The BIT Mesra College Dispensary provides comprehensive medical services to students, faculty, and staff members. Our modern healthcare facility is equipped with:

- **24/7 Emergency Services**
- **Qualified Medical Professionals**
- **Modern Medical Equipment**
- **Digital Health Records**
- **Online Appointment System**

## 🚀 System Features

### 👨‍🎓 Student Portal
- **Dashboard**: Overview of health status and upcoming appointments
- **Book Appointment**: Schedule appointments with doctors
- **My Appointments**: View and manage appointment history
- **Emergency SOS**: Send emergency alerts to medical staff
- **Prescriptions**: Access digital prescriptions
- **AI Chatbot**: Get instant medical assistance
- **Profile Management**: Update personal and medical information

### 👨‍⚕️ Admin Portal
- **Doctor Management**: Manage doctor schedules and availability
- **Queue Management**: Monitor patient queues
- **Analytics**: Health statistics and reports
- **Emergency Alerts**: Monitor and respond to SOS alerts
- **QR Scanner**: Scan student IDs for quick access
- **Leave Management**: Handle doctor leave requests

## 🛠️ Technical Stack

### Frontend
- **React.js** - Modern UI framework
- **Material-UI** - Component library
- **React Query** - Data fetching and caching
- **React Router** - Navigation
- **Axios** - HTTP client

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB** - Database
- **Mongoose** - ODM
- **JWT** - Authentication
- **Socket.io** - Real-time communication

### AI Integration
- **Hugging Face** - Free AI models
- **OpenAI** - GPT integration
- **Cohere** - Alternative AI service
- **Local Fallback** - Rule-based responses

## 📋 Installation Guide

### Prerequisites
- Node.js (v14+)
- MongoDB (v4+)
- npm or yarn

### Backend Setup
```bash
cd /Users/anshusmac/Desktop/DOC
npm install
npm start
```

### Frontend Setup
```bash
cd /Users/anshusmac/Desktop/DOC/client
npm install
npm start
```

### Environment Variables
```env
NODE_ENV=development
PORT=5001
MONGODB_URI=mongodb://localhost:27017/college-dispensary
JWT_SECRET=college_dispensary_jwt_secret_key_2024
JWT_EXPIRE=7d
```
